### Admin Package

Admin package lokal sudah dibuat di:

```txt
packages/admin/
```

Route admin:

- `/admin`
- `/admin/login`
- `/admin/logout`
- `/admin/dashboard`
- `/admin/settings`
- `/admin/company-profile`
- `/admin/products`
- `/admin/management`

Auth sementara dari `.env`:

```env
ADMIN_USERNAME=admin
ADMIN_PASSWORD=change-me-admin
```

Setting admin tersimpan di:

```txt
storage/admin/settings.json
```

Admin sudah dites:

- Login page 200.
- Dashboard redirect ke login jika belum login.
- Login berhasil dengan credential `.env`.
- Submit settings berhasil update JSON.
